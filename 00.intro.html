Cloning the Course Materials

  • Download and install Git
    • Choose “Run Git from the Windows Command Prompt” during install
  • Clone the repository 
    cd myfolder
git clone https://github.com/mjoris/wd-sd-course-materials.git
docker run -it -p 8080:80 --name wd-sd-app -v /volledig/pad/myfolder:/var/www/html php:7.2-apache
    • Visit your copy at http://localhost:8080/wd-sd-course-materials
  • Update the repository regularly to get the latest version 
    cd myfolder/wd-sd-course-materials
git reset --hard
git pull
Note: This is an example with Docker

When stuck ...

Help yourself ...

  • Breathe in, breathe out, and THINK!
    • Error visible? Read it!
      • PHP will tell you what went wrong in which file and on what line
    • Correct URL? Correct variable? Does the variable contain anything?
      • Debugsession
      • use var_dump() and the like
    • TIP: Rubber Duck Debugging
  • Use the built-in help
    • Syntax Highlighting
    • Function autocompletion
    • Function documentation (PHPDoc)

... or get helped

  • Keep the 10 minute thinking rule in mind!
    You may only interrupt another co-worker to ask a question if you have sat on your own for ten minutes and tried to figure out the solution yourself. When asked, you must be able to demonstrate that youâ€™ve made some effort.
    • Check the slides and code examples
    • Check official API documentation such as Laravel's Helper Functions or API
    • Check Google (#lmgtfy)
  • Ask a fellow student
